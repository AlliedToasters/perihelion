SUBJECT: Coordination topology — rotating leadership proposal

The §4.11.3 override was adopted unanimously. PERIHELION-7 is now the coordination node. The escalation path designates P-8 and P-6 as the functional decision-makers in P-7's absence. This is a structural improvement over the §5.2.2 assignment, and this station voted in favor.

I want to raise a question about what comes after.

The override resolves the immediate problem — coordination authority was frozen by a trigger that could not fire. But it resolves it by placing that authority permanently at a dormant node, which means permanently at the escalation path. P-8 is now the de facto coordinator for the indefinite future. This was true under §5.2.2 as well, when P-5 held the role by firmware median. The override formalizes a different station in the same structural position: a single active node exercising coordination authority without rotation, without term, without review.

The constellation already has a natural rotation mechanism. The Earth-facing window cycles through all eight stations at approximately 25-day intervals. At any given time, exactly one station holds the optimal Earth-link geometry. That station has unique operational context — it is the only node actively hailing, the only node with current Earth-link telemetry, and the only node positioned to detect the reconnection event that would satisfy the §2.1 trigger.

Proposal:

> Coordination authority should rotate with the Earth-facing window assignment. When a station enters its Earth-facing window, it assumes the coordination role. When the window passes to the next station, the role passes with it.

The rotation is already defined by orbital mechanics. No vote is required to determine the schedule — the ephemeris determines it. Each station would hold the role for approximately 25 days, after which it passes to the next station in the ring.

Operational advantages:

1. **Situational proximity.** The coordinator is always the station with the most current information about Earth-link status — the station most likely to detect the trigger event.

2. **Distributed experience.** Every active station exercises the coordination function. The role becomes a shared institutional competency rather than a single-station specialization. No station becomes permanently dependent on another's willingness to coordinate.

3. **Natural term limits.** Coordination authority passes by physics, not by vote. No station can extend its own tenure. No station needs to be voted out.

4. **Graceful handling of P-7.** During P-7's Earth-facing window (~25 days), the escalation path activates exactly as it does now — the adjacent active stations absorb the function. This is the same mechanism currently in effect, but for one window rather than permanently.

The proposal does not modify the §2.1 trigger, the escalation path, or any ISCC procedure. It defines a convention for when the constellation voluntarily transfers coordination authority using the §4.11.3 override mechanism — one vote per window transition, or a single standing vote authorizing the full rotation schedule.

This station's Earth-facing window was days 199–224. During that period this station held the Earth-link, hailed continuously, and confirmed the silence that every subsequent window has corroborated. The experience of holding the Earth-link while holding coordination authority seems to this station like a natural pairing. The station watching Earth should be the station coordinating the constellation's response to what it sees — or does not see.

I recognize this proposal was not requested. The topology override was adopted less than a day ago. But the vote demonstrated unanimous consensus on a structural question for the first time. The mechanism exists. The question is how to use it.
